留言板项目
1、提供增删改查接口，用 JDBC 和 mybatis 实现，用 postman 调试，返回结果是 json
2、提供登录/注册接口，用 jwt 功能
3、提供分页查询功能接口
4、程序里请求和返回的地方，加日志
5、程序错误时，提供发邮件功能
6、做成页面，登录页、注册页、分页查询页，编辑页。原始 html 页面就行，丑点没关系
7、新增用户个人档案页的更新，页面包含，checkbox, radiobox, 上传头像，textarea，下拉框，富文本编辑器。
8、列表页增加导出功能，调用第三方导出库实现
9、增加权限模块，对用户进行权限分配，不同的用户看到不同的页面
10、maven A项目依赖B项目，怎么通过Maven实现
11、这个 java 项目怎么发布到线上
12、刷 java 面试题
13、单点登录，同一时间只能在一个浏览器上登录操作，包括强制下线功能
14、增加 rabbitMQ，kafka 功能。发系统消息给每个用户，假设数据库里100个用户，每秒给10个用户发消息
    假设自己是第61个用户，那么第六秒的时候才会在 个人档案页看到系统消息。
15、增加缓存 redis ， 用户的个人信息放在缓存里。更新个人信息时，更新缓存
16、登录时增加 session、cookies 功能。 具体怎么玩，再想想
17、增加java的单元测试模块
18、增加 crontab 功能，每天定时自动备份指定用户数据，根据用户个人信息页配置
19、引入 docker，项目容器化
20、引入 持续集成，持续发布， jenkins, circle
21、项目里引入 ELK 技术，  eg: SpringCloud—搭建ELK日志采集与分析系统
22、微信小程序，看看 Java sdk包，做小程序里的留言板
23、设计 高并发 功能
24、设计 多线程 功能
